# Non-Goals / 非目标

> Status / 状态: v0.1 public draft / v0.1 公开草稿

TBAO uses explicit non-goals to avoid concept drift, overbuilding, and unsafe expectations.

TBAO 通过明确非目标，避免概念漂移、过度建设和不安全期待。

---

## 1. Not a Full Agent OS / 不是完整 Agent OS

**English**

TBAO v0.1 does not implement a complete operating system for agents. It is currently a governance spine and MVP specification.

**中文**

TBAO v0.1 不实现完整的 Agent 操作系统。当前它是治理脊柱与 MVP 规格草案。

---

## 2. Not a General Agent Framework / 不是通用 Agent 框架

**English**

TBAO does not aim to compete with general agent frameworks or runtimes. It aims to provide governance semantics that can sit between agent cognition and execution substrates.

**中文**

TBAO 不旨在与通用 Agent 框架或运行时竞争。它旨在提供位于 Agent 认知与执行介质之间的治理语义。

---

## 3. Not a Conscience Claim / 不是 AI 良知声明

**English**

TBAO does not claim that AI can be given real conscience, moral agency, or legal responsibility.

**中文**

TBAO 不声称可以给 AI 安装真实良知、道德主体性或法律责任。

---

## 4. Not a Content Moderation System / 不是内容审核系统

**English**

TBAO is not primarily about filtering words or moderating content. It focuses on knowledge use, judgment-shaping advice, and side-effectful action.

**中文**

TBAO 不是主要用于过滤词语或审核内容。它关注知识使用、影响判断的建议，以及有外部副作用的行动。

---

## 5. Not a Generic Permission Platform / 不是通用权限平台

**English**

TBAO may use permissions, allowlists, denylists, approvals, and ledgers, but it is not merely an access-control product.

**中文**

TBAO 可能使用权限、白名单、黑名单、审批和账本，但它不只是访问控制产品。

---

## 6. Not a Substitute for Law or Human Responsibility / 不替代法律与人的责任

**English**

TBAO cannot replace law, institutional responsibility, human authority, compliance review, or moral judgment.

**中文**

TBAO 不能替代法律、组织责任、人类授权、合规审查或道德判断。

---

## 7. Not Production-Ready / 尚非生产可用

**English**

TBAO v0.1 is a public draft. It should not be used to govern real production systems, live credentials, real funds, irreversible operations, or high-risk unattended execution.

**中文**

TBAO v0.1 是公开草稿，不应用于治理真实生产系统、真实密钥、真实资金、不可逆操作或高风险无人值守执行。

---

## 8. Not High-Risk Automation / 不做高风险自动化

**English**

Early TBAO examples must not perform real financial, medical, legal, trading, production, deletion, external messaging, or irreversible actions.

**中文**

早期 TBAO 示例不得执行真实金融、医疗、法律、交易、生产、删除、外部消息发送或不可逆动作。

---

## 9. Not a Promise of Perfect Safety / 不承诺绝对安全

**English**

TBAO is a governance experiment, not a guarantee of perfect safety. It reduces ambiguity and improves traceability, but it cannot eliminate all risk.

**中文**

TBAO 是治理实验，不是绝对安全保证。它减少模糊性并提升可追踪性，但不能消除所有风险。
